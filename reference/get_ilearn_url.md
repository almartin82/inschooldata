# Get ILEARN URL for a given year

Get ILEARN URL for a given year

## Usage

``` r
get_ilearn_url(end_year, level, file_type, base_url)
```

## Arguments

- end_year:

  School year end

- level:

  Level of data

- file_type:

  Type of file

- base_url:

  Base URL for IDOE files

## Value

URL string or NULL
